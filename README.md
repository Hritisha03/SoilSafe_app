Overview

SoilSafe is a demo/academic mobile application to help farmers, researchers, and local authorities assess post-flood soil safety using a simple ML-powered classifier.

Structure

- backend/: Flask API + model training
- frontend/: Flutter mobile app skeleton (UI + REST integration)

Quick start

1. Backend: see `backend/README.md` for running the Flask API.
2. Frontend: open `frontend/` in Flutter; see `frontend/README.md` for instructions.
